== Description ==
This simple plugin lets you customize your website scrollbar with colors and design.
Your can design your scrollbar to match with your theme.
You can also purchase this plugin to get support.

If any issues regarding this plugin mail me @: sandeep.tete2013@gmail.com


== Installation ==
- Upload zip file.
- Activate the plugin.
- Go to Appearance -> customize.
- Find custom scrollbar designer.
-Modify the settings and you are done.
